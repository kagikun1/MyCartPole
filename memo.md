# 備忘録

ssh接続手順

## 鍵ペア作成  
`cd ~/.ssh`  
`ssh-keygen -t rsa`  
[アップロード画面](https://github.com/settings/keys)

## 接続確認  
`ssh -T git@github.com`  
[参考サイト](https://qiita.com/shizuma/items/2b2f873a0034839e47ce)  

## git push 手順  
`git add <filename>`  
`git commit`  
`git push origin <branch>`
